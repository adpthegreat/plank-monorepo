# Introduction

Welcome to the Plank programming language documentation.

Plank is a programming language for writing smart contracts on the EVM. It compiles to EVM bytecode via Sensei IR and provides modern language features like compile-time evaluation, a module system, and first-class functions.

Plank is in its early stages. If you run into bugs, have ideas, or want to get involved, open an issue on [GitHub](https://github.com/plankevm/plank-monorepo) or join the community chat.
